# AfriTalent — Plateforme Freelance Tech Afrique

## Informations du projet
- **Étudiant :** SOW Amadou
- **Classe :** [CS]
- **Établissement :** Groupe ISI
- **Année :** 2025-2026

## Description
AfriTalent est une plateforme fictive de mise en relation entre freelances tech
africains et entreprises. Ce projet vitrine a été réalisé dans le cadre du
Semestre 2 en développement web.

## Lien du site en ligne
[Voir le site sur GitHub Pages](https://TON-PSEUDO.github.io/SOW-Amadou-AfriTalent)

## Pages du site
- **Accueil** — Hero, statistiques animées, catégories, témoignages
- **Freelances** — Catalogue filtrable de 9 profils
- **Tarifs** — 3 plans tarifaires + FAQ accordion
- **À propos** — Histoire, équipe, valeurs, chiffres clés
- **Contact** — Formulaire validé + carte Google Maps

## Technologies utilisées
- HTML5 sémantique
- CSS3 (Flexbox, Grid, Bento Grid, variables CSS, animations)
- Bootstrap 5 (navbar, carousel, accordion, cards)
- Bootstrap Icons
- Google Fonts (Poppins + Inter)
- JavaScript vanilla (dark mode, compteurs, filtrage, validation)
- Git & GitHub Pages

## Fonctionnalités JavaScript
1. Dark mode avec localStorage (persiste entre les pages)
2. Navbar dynamique au scroll (effet shrink + ombre)
3. Bouton retour en haut (smooth scroll)
4. Compteurs animés au scroll (IntersectionObserver)
5. Animations fade-in des sections (IntersectionObserver)
6. Filtrage dynamique des freelances par catégorie
7. Validation complète du formulaire de contact (regex email, 20 car. min)

## Lancer le projet localement
1. Clone le dépôt :
   git clone https://github.com/TON-PSEUDO/SOW-Amadou-AfriTalent.git
